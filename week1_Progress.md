**TalentView – HR Analytics Project**
TalentView is a Power BI project designed to analyze workforce composition, employee attrition, and hiring effectiveness using HR data.  
The project is being developed in a structured, week-wise manner to reflect real-world analytics workflows.

**Week 1 – Workforce Overview**

In Week 1, the focus was on building a strong foundation for HR analytics.
**What was done:**
1. Collected and prepared HR datasets (employee, department, demographics, headcount, separations)
2. Cleaned and anonymized data using Power Query
3. Built a proper star-schema data model in Power BI
4. Created core DAX measures for:
  - Total Headcount
  - Average Tenure (Years)
5. Designed a professional Workforce Overview dashboard showing:
  - Headcount by department
  - Employee demographics (gender, age group, ethnicity)
  - Interactive filters for easy exploration

The goal of Week 1 was to provide a clear, company-wide snapshot of the workforce.
